## Gin RESTful

### 项目简介

使用`Gin`框架搭建`RESTful`风格的基础脚手架，开箱即用，快速实现项目的业务功能开发

- docs：
  - 项目文档目录；包含`API`接口文档等...
- restapi：
  - 后端`API`接口目录
- web：
  - 前端目录
